# YAML Input Schema - anonymous-admin

> LoG.ai Layer 3 code-generation input for the **anonymous-admin** micro-app. Companion to
> [`3.field-spec.md`](3.field-spec.md) and [`3.framework-mapping.md`](3.framework-mapping.md).
> Only template-defined properties are used. Framework spelling: `collapsable` (NOT collapsible).
> `adminReportDoc` is `autoSave: true` → the Context bootstrap is mandatory (BRD §8.2, D-L3-2).
>
> **No admin Doc binds any reporter-identity field** (`reporterId`/`contactMethod`/`contactValue`/
> reporter-create audit). Every `reports` read flows through `loadReportsForAdmin` /
> `loadReportForAdmin` which apply `{ projection: adminProjection }` (ER-A3). Routing only through
> `resolveAssignees` (D17).
